# 《断锚求生》拼装船锚玩法策划案

版本：v0.1  
日期：2026-07-03  
文档目标：将“船锚断裂后，用一网捞上来的杂物临时拼出船锚并下锚求生”的玩法整理为可开发、可调参、可验证的一期方案。

---

## 1. 项目概述

### 1.1 核心一句话

船锚突然断裂，玩家需要在限定时间内用随机捞上来的物品拼出临时船锚，并让它在 2D 侧视场景中经历入水、下沉、触底三个受力阶段，最终判断船是否被成功稳住。

### 1.2 核心乐趣

- 用“不像锚的东西”拼出一个勉强能用的锚。
- 在有限时间和有限材料中做结构取舍。
- 看自己拼出的怪东西下水后被物理系统逐步检验。
- 失败不是直接数值判定，而是能看见哪里被扯开、哪里承压过大、哪里抓地不足。

### 1.3 一期目标

一期做成 2D 侧视物理拼装玩法，所有拼装、检测、受力和表现都以 2D 平面为准。

玩家在拼装阶段把几何物品摆放到工作区，系统根据 2D 图形边框紧贴面积、材料、重量、形状、粘合系数生成临时连接。物品需要区分“视觉图形轮廓”和“实际碰撞箱”：连接判定优先使用图形轮廓，物理碰撞和运动稳定性使用碰撞箱。下锚后进入自动模拟阶段，受力阶段只分为入水、下沉、触底，分别计算入水冲击、下沉过程中的水流横向力/船锚绳力/重力，以及触底后的水流衰减、海底摩擦和海底支撑。

### 1.4 一期不做

- 不做任何 3D 拼装或 3D 受力模拟。
- 不做真实流体模拟。
- 不做复杂材料断裂网格。
- 不做玩家在下锚阶段继续手动修补。
- 不做多人对战。

---

## 2. 局内核心循环

1. 开局生成本局船只重量、海况、目标锚固需求。
2. 一网捞起一批随机材料。
3. 玩家在限定时间内拼装临时船锚。
4. 玩家提交，系统检查是否满足基础结构下限。
5. 下锚，拼装体进入阶段模拟。
6. 系统展示结构受力、粘合面积衰减、部件脱落、触底抓地过程。
7. 结算成功、勉强成功或失败，并给出主要失败原因。
8. 发放下一局或关卡奖励，解锁新材料、预制形状或船只条件。

## 3. 局内流程

### 3.1 开局参数随机

每局开始生成以下参数：

| 参数 | 说明 | 一期建议范围 |
|---|---|---|
| 船只重量 | 影响所需锚固力和绳索拉力 | 800kg - 5000kg |
| 船体迎流面积 | 影响水流对船的拉扯 | 低 / 中 / 高 |
| 海况等级 | 影响水流力、波动拉力、入水扰动 | 1 - 5 |
| 水深 | 影响下沉时间和绳索长度 | 8m - 40m |
| 海底类型 | 影响抓地系数 | 沙地 / 泥地 / 碎石 / 岩底 |
| 拼装时间 | 玩家可操作时间 | 45s - 120s |
| 材料池等级 | 决定可随机到的物品种类 | 1 - 5 |

### 3.2 材料随机

开局“一网捞上来”的材料由材料池随机生成。

材料应同时具备玩法属性和直观外观。玩家不需要理解复杂公式，但需要能通过 UI 看懂大致特性。

材料基础字段：

| 字段 | 说明 |
|---|---|
| 材料 ID | 配置唯一标识 |
| 名称 | 展示名称 |
| 形状类型 | 矩形、圆形、三角形、钩形、链条形、异形 |
| 尺寸 | 影响图形边框贴合面积、阻力、抓地 |
| 视觉图形轮廓 | 用于玩家看到的 2D 外形和连接贴合判定 |
| 实际碰撞箱 | 用于物理碰撞、下沉运动和海底接触，可与视觉图形轮廓不同 |
| 重量 | 影响下沉、绳索拉力、连接承压 |
| 密度 | 影响浮力与是否容易突破水面 |
| 粘合系数 | 影响贴合后的连接强度 |
| 表面粗糙度 | 影响边框贴合粘合与触底抓地 |
| 抗拉强度 | 部件自身被拉裂的阈值 |
| 抗剪强度 | 边框贴合连接被错位拉开的阈值 |
| 水阻系数 | 下沉阶段受力 |
| 抓地系数 | 触底后提供锚固力 |
| 特殊标签 | 尖锐、钩挂、吸水、易碎、漂浮、柔性 |

一期材料示例：

| 材料 | 类型 | 玩法定位 |
|---|---|---|
| 生锈铁块 | 重物 | 下沉稳定，粘合一般 |
| 破木板 | 轻质板材 | 面积大，易浮，粘合中等 |
| 渔网团 | 柔性物 | 可增加接触和缠绕，抗拉低 |
| 铁钩 | 钩形件 | 抓地强，接触面小 |
| 石块 | 重物 | 下沉强，形状不规则，粘合差 |
| 船链碎段 | 链条件 | 可作为延伸结构，局部受力高 |
| 橡胶轮胎 | 环形件 | 缓冲好，抓地弱，水阻大 |
| 破铁锚爪 | 稀有件 | 抓地强，重量高 |
| 塑料桶 | 漂浮件 | 容易失败，但可做大面积结构 |
| 水泥块 | 高重量件 | 下沉强，接触面稳定 |

## 4. 拼装阶段

### 4.1 拼装目标

玩家需要在限定时间内把随机材料组合成一个临时船锚。一个可用船锚至少需要满足：

- 有足够总重量，能够突破水面张力并沉入水中。
- 有足够整体结构强度，不会在下锚初期直接散架。
- 有可连接船绳的主受力点。
- 有至少一个可抓地结构，能在触底后产生锚固力。

### 4.2 操作方式

- 玩家从材料栏拖拽物品到拼装区。
- 物品可旋转、移动、叠放。
- 物品边框进入贴合冗余范围时，系统显示可连接的边框贴合区域。
- 松手后，系统根据 2D 图形边框紧贴面积与材料属性生成连接，不要求实际碰撞箱必须发生接触。
- 可撤销上一步。
- 可删除已放置物品，删除后材料回到材料栏。
- 可选择船绳连接点。

### 4.3 2D 图形边框紧贴粘住

一期采用“边框贴近即尝试粘连”的规则。玩家看到的是 2D 图形轮廓之间的贴合，不再用两个碰撞箱的重叠面积作为连接依据。

系统为每个物品维护两套形状：

| 形状 | 用途 |
|---|---|
| 视觉图形轮廓 | 连接判定、贴合高亮、玩家理解结构 |
| 实际碰撞箱 | 运动碰撞、海底支撑、物理稳定性 |

贴合判定允许一定冗余空间。两个物品的实际碰撞箱可以没有接触，只要图形轮廓边框距离足够近，且贴合段长度/面积达到阈值，就可以生成临时连接。

连接生成条件：

1. 两个物品的 2D 视觉图形边框进入贴合冗余范围。
2. 边框紧贴面积达到最小有效贴合面积。
3. 两个物品均允许参与粘合。
4. 两个物品不处于互斥材料组合。

连接强度计算：

```text
连接强度 = 边框紧贴面积
        * 材料A粘合系数
        * 材料B粘合系数
        * 表面匹配系数
        * 形状稳定系数
        * 贴合冗余修正
        * 难度修正
```

贴合冗余修正用于让判定不那么苛刻：

```text
贴合冗余修正 = clamp01(1 - 边框间距 / 最大贴合冗余距离)
```

边框越接近，修正越高；只要仍在冗余距离内，就可以形成弱连接。

表面匹配系数示例：

| 组合 | 系数 |
|---|---:|
| 铁 + 铁 | 1.00 |
| 铁 + 木 | 0.75 |
| 木 + 木 | 0.90 |
| 石 + 铁 | 0.55 |
| 橡胶 + 铁 | 0.80 |
| 塑料 + 任意 | 0.40 |

### 4.4 下限判断预制

为避免玩家拼出完全不可模拟的结构，一期在提交时做基础下限判断。

提交下限：

| 判断项 | 通过条件 | 失败提示 |
|---|---|---|
| 总重量 | 总重量 >= 本局最低下沉重量 | 太轻了，可能漂在水面上 |
| 主连接点 | 已选择船绳连接部件 | 需要选择船绳绑在哪里 |
| 结构连通 | 船绳连接部件能连通到至少 70% 总重量 | 绑点和主体没有连住 |
| 贴合连接 | 至少存在 2 个有效边框贴合连接 | 结构太松散 |
| 抓地结构 | 至少 1 个部件抓地系数 > 0 | 触底后没有能抓住海底的部分 |

预制辅助：

- 系统提供“船绳绑点”预制标记，玩家可放在任意一个部件上。
- 系统提供“推荐重心区”提示，只在新手关开启。
- 系统不直接自动修正结构，只提示风险。

### 4.5 拼装阶段 UI

主要区域：

| 区域 | 内容 |
|---|---|
| 顶部局势栏 | 船重、海况、水深、海底类型、剩余时间 |
| 左侧材料栏 | 本局捞到的材料 |
| 中央拼装区 | 拖拽、旋转、叠放、显示连接 |
| 右侧结构评估 | 总重量、预计下沉、预计强度、预计抓地 |
| 底部操作栏 | 撤销、清空、提交下锚 |

结构评估只显示等级，不显示完整公式：

- 总重量：不足 / 勉强 / 充足 / 过重
- 结构强度：脆弱 / 可用 / 牢固
- 抓地能力：无 / 弱 / 中 / 强
- 水阻风险：低 / 中 / 高

## 5. 下锚模拟总规则

### 5.1 阶段划分

下锚后进入自动模拟，受力阶段只按以下三段依次执行：

1. 入水阶段：短时间冲击检验，水面对船锚下方材料施加较大的向上冲力。
2. 下沉阶段：在水中持续模拟水流横向力、船锚绳力、重力、浮力、水阻和结构连接损伤。
3. 触底阶段：接触海底后降低水流横向影响，增加海底摩擦、海底支撑、抓地与滑动判定。

结算不是独立受力阶段，只在触底稳定判定结束或结构提前失败后触发。

### 5.2 模拟对象

每个材料部件都是独立受力对象。

每个连接也是独立对象，保存以下数据：

| 字段 | 说明 |
|---|---|
| 连接 ID | 唯一标识 |
| 部件 A | 连接一端 |
| 部件 B | 连接另一端 |
| 初始边框紧贴面积 | 拼装完成时的有效贴合面积 |
| 当前边框紧贴面积 | 受力后剩余有效贴合面积 |
| 最大连接强度 | 由初始边框紧贴面积与材料决定 |
| 当前连接强度 | 随当前边框贴合面积变化 |
| 受拉累计损伤 | 拉扯造成 |
| 受剪累计损伤 | 错位造成 |
| 是否断开 | 当前连接是否失效 |

### 5.3 边框紧贴面积逐渐减小

连接受力超过阈值时，不立即断裂，而是逐渐减少边框紧贴面积。这里的面积是视觉图形轮廓的贴合面积，不是实际碰撞箱重叠面积。

```text
面积衰减量 = max(0, 当前受力 - 当前连接强度)
          * 衰减速度系数
          * 时间步长
```

当当前边框紧贴面积低于断开阈值时，连接断开。

```text
若 当前边框紧贴面积 <= 初始边框紧贴面积 * 最小保留比例
则 连接断开
```

推荐参数：

| 参数 | 一期默认值 |
|---|---:|
| 衰减速度系数 | 0.08 |
| 最小保留比例 | 0.15 |
| 最小有效贴合面积 | 12 px² |
| 最大贴合冗余距离 | 6 px |
| 连接断开延迟 | 0.15s |

## 6. 入水阶段

### 6.1 阶段目标

判断拼装体是否能进入水中，并给予第一次冲击检验。入水冲击来自水面对船锚下方材料的短时间强反作用力，重点检验底部连接是否会被瞬间顶开。

### 6.2 关键受力

- 重力
- 浮力
- 水面张力
- 入水冲击力：从船锚下方向上作用，持续时间短，数值较大
- 初始绳索拉力：只作为轻量扰动，不单独形成受力阶段

入水冲击力简化：

```text
底部入水冲力 = 入水冲击基础值
          * 海况修正
          * 入水速度修正
          * 底部迎水面积
```

受力对象：

- 只作用于船锚下方迎水的材料部件。
- 优先转化为连接剪切力和局部压缩/顶开表现。
- 持续时间建议 0.15s - 0.35s，超过后进入下沉阶段。

### 6.3 突破水面张力

拼装体需要满足下沉驱动力大于水面阻力。

```text
下沉驱动力 = 总重力 - 总浮力
水面阻力 = 水面张力基础值 * 接触水面宽度 * 海况修正
```

通过条件：

```text
下沉驱动力 > 水面阻力
```

失败表现：

- 拼装体浮在水面上。
- 船绳绷紧后持续拖拽，结构被水面反作用力和船体拉力撕开。
- 结算为“未能入水”。

可调参数：

| 参数 | 说明 |
|---|---|
| surface_tension_base | 水面张力基础值 |
| surface_width_weight | 水面接触宽度权重 |
| sea_state_surface_mul | 海况对水面阻力修正 |
| buoyancy_mul | 浮力整体修正 |

## 7. 船锚下沉阶段

### 7.1 阶段目标

拼装体在水中下沉，系统统一计算水流横向力、船锚绳力、重力、浮力、水阻、翻滚、连接受力和部件脱落。

### 7.2 下沉速度

```text
净下沉力 = 总重力 - 总浮力 - 水阻
水阻 = 0.5 * 水密度 * 速度² * 投影面积 * 水阻系数
```

一期可简化为：

```text
下沉速度 += (总重量系数 - 浮力系数 - 水阻系数) * dt
```

### 7.3 部件独立受力

每个部件受到：

- 自身重力
- 浮力
- 水阻
- 水流横向力
- 来自连接的拉力和剪切力
- 来自船绳的传导力

当部件与主体连接断开后：

- 若密度大于水，继续下沉。
- 若密度小于水，上浮或悬浮。
- 脱落部件不再贡献最终锚固力。

### 7.4 翻滚与姿态

一期不做完整刚体链复杂模拟，可采用简化姿态规则：

- 计算整体重心。
- 计算水阻中心。
- 重心与水阻中心偏移越大，翻滚风险越高。
- 翻滚风险转化为连接剪切力。

```text
翻滚剪切力 = 重心水阻中心距离 * 下沉速度 * 翻滚系数
```

## 8. 下沉阶段 - 船锚绳受力

### 8.1 阶段目标

船锚绳是下沉阶段的一类持续受力来源，不单独形成阶段。锚体越重、水流越强、船越重，绳索拉力越大。

### 8.2 绳索连接点

玩家在拼装阶段选择一个部件作为船绳连接点。

该部件承受主拉力，并通过连接网络向其他部件传导。

### 8.3 绳索拉力计算

```text
基础拉力 = 船只重量 * 船体受水流系数 * 海况系数
动态拉力 = 船体摇摆系数 * 波动强度
绳索总拉力 = 基础拉力 + 动态拉力
```

绳索拉力传导：

- 先作用于绑点部件。
- 按连接图向相邻部件分配。
- 路径越短，承受力越高。
- 边框贴合面积越大，分担能力越强。

一期简化：

```text
连接承受拉力 = 绳索总拉力 * 连接路径权重 / 有效承力连接数量
```

### 8.4 绑点失败

如果绑点部件断开：

- 船绳脱离主体。
- 锚体失效。
- 结算为“绑点被拉断”。

---

## 9. 下沉阶段 - 水流横向受力

### 9.1 阶段目标

水流横向力是下沉阶段的一类持续受力来源，不单独形成阶段。水流会持续冲击锚体和船体，使结构产生横向拉扯。

### 9.2 水流对锚体

```text
水流力 = 水流速度² * 部件投影面积 * 部件水阻系数 * 海况修正
```

水流主要造成：

- 横向偏移
- 连接剪切
- 翻滚
- 柔性材料摆动损伤

### 9.3 水流对船

水流对船的作用通过船绳传给锚体。

```text
船体水流拉力 = 船体迎流面积 * 水流速度² * 船体系数
```

### 9.4 判定间隔优化

如果逐帧计算所有连接受力导致卡顿，可采用间隔判定。

优化方案：

| 方案 | 说明 |
|---|---|
| 固定判定间隔 | 每 0.1s 计算一次连接损伤 |
| 分层更新 | 高风险连接每帧更新，低风险连接降频 |
| 事件触发 | 只有受力突变、碰撞、姿态变化时重算 |
| 连接数量上限 | 一期限制有效连接不超过 50 个 |
| 低权重合并 | 小碎片可合并成装饰碎片，不参与完整受力 |

推荐一期：

- 连接受力每 0.1s 更新一次。
- 视觉表现每帧插值。
- 当连接受力超过 80% 强度时，提升到每帧检测。

## 10. 触底阶段

### 10.1 阶段目标

拼装体接触海底后，进入触底阶段。此时水流横向力减少，系统增加海底摩擦、海底支撑和抓地判定，判断是否能产生足够锚固力来稳住船。

### 10.2 海底类型

| 海底类型 | 抓地特性 |
|---|---|
| 沙地 | 普通抓地，尖锐件表现好 |
| 泥地 | 重物容易陷入，抓地稳定 |
| 碎石 | 不规则件表现好，光滑件容易滑 |
| 岩底 | 钩形件表现好，普通重物容易滑 |

### 10.3 抓地力计算

触底后新增三类海底作用：

| 作用 | 说明 |
|---|---|
| 水流衰减 | 锚体贴近或接触海底后，横向水流力按海底遮挡和埋入程度降低 |
| 海底摩擦 | 与海底接触的碰撞箱产生摩擦，抵抗横向拖拽 |
| 海底支撑 | 海底向上支撑接触部件，减少继续下沉并改变受力路径 |

```text
触底后水流力 = 原水流横向力 * 触底水流衰减系数
海底摩擦力 = 法向支撑力 * 海底摩擦系数 * 材料粗糙度
海底支撑力 = 接触深度 * 海底硬度系数
```

```text
部件抓地力 = 部件重量
          * 抓地系数
          * 海底匹配系数
          * 接触角度系数
          * 埋入深度系数
```

总抓地力：

```text
总抓地力 = 所有仍连接到船绳路径上的有效部件抓地力之和
```

只有仍与船绳连接网络连通的部件才计入锚固力。

### 10.4 触底后的稳定判定

触底后持续模拟一段稳定时间，例如 8 秒。

每秒判断：

- 总抓地力是否大于船绳拉力。
- 连接是否继续衰减。
- 是否有关键部件被拉脱。
- 锚体是否整体滑动。
- 海底摩擦与支撑是否足以抵消剩余横向水流力。

成功条件：

```text
总抓地力 >= 船绳总拉力 * 安全系数
并且 稳定时间内主体未断开
```

推荐安全系数：

| 难度 | 安全系数 |
|---|---:|
| 新手 | 0.90 |
| 普通 | 1.00 |
| 困难 | 1.20 |
| 风暴 | 1.50 |

## 11. 结算规则

### 11.1 结算档位

| 档位 | 条件 | 表现 |
|---|---|---|
| 完美稳住 | 稳定时间结束，结构损伤低于 20% | 船只停止漂移 |
| 勉强稳住 | 稳定时间结束，结构损伤 20%-70% | 船只仍晃动，但未漂走 |
| 结构失败 | 关键连接断开 | 锚体散架 |
| 入水失败 | 未突破水面张力 | 漂在水面 |
| 下沉失败 | 下沉中主体解体 | 部件四散 |
| 抓地失败 | 触底后总抓地力不足 | 锚体被拖走 |
| 绑点失败 | 船绳连接部件断开 | 船绳空拉 |

### 11.2 失败原因展示

结算时显示最多 3 个主要原因：

- 总重量不足。
- 漂浮材料过多。
- 绑点承力不足。
- 边框贴合面积太小。
- 材料粘合系数太低。
- 抓地结构不足。
- 水阻过大导致下沉中翻滚。
- 船太重，绳索拉力超过结构上限。

### 11.3 奖励

一期可采用轻量成长：

- 通关获得金币。
- 根据表现获得材料图鉴经验。
- 解锁新材料池。
- 解锁新的预制绑点外观或提示工具。

## 14. 程序实现建议

### 14.1 一期技术路线

推荐使用 2D 物理引擎：

- Unity 2D + Rigidbody2D + Collider2D
- 或 Web 端 Phaser + Matter.js

如果目标是快速验证玩法，优先做 Unity 2D 原型。

### 14.2 核心模块

| 模块 | 职责 |
|---|---|
| RoundGenerator | 生成船重、海况、材料 |
| BuildController | 拼装拖拽、旋转、提交 |
| SnugFitDetector | 检测 2D 图形边框紧贴面积 |
| JointBuilder | 根据边框贴合和材料生成连接 |
| ForceSimulator | 阶段受力计算 |
| JointDamageSystem | 边框贴合面积衰减和断开 |
| AnchorEvaluation | 下限检查和结算 |
| ResultPresenter | 结算表现和失败原因 |

### 14.3 连接图

拼装体应维护一张连接图：

- 节点：材料部件。
- 边：边框贴合连接。
- 船绳绑点：指定节点。
- 有效主体：从绑点出发可遍历到的所有节点。

结算时只统计有效主体，不统计断开的孤立部件。

### 14.4 边框贴合面积计算简化

一期可用以下方式降低实现成本：

- 矩形、圆形、三角形使用视觉图形轮廓的边缘采样。
- 异形件使用多个简单 collider 组合。
- 贴合面积可用“贴合段长度 * 等效厚度”近似，不直接取 collider 交集面积。
- 实际碰撞箱只负责运动碰撞，允许与视觉图形轮廓存在差异。
- 视觉上显示为高亮贴合带，不要求完全精确。

## 16. 原型开发里程碑

### 16.1 第 1 周：拼装原型

- 完成材料拖拽、旋转、放置。
- 完成 2D 图形边框贴合检测。
- 完成连接生成。
- 完成提交下限判断。

验收标准：

- 玩家能用 5-8 个物品拼出结构。
- 系统能显示哪些部件被连接。
- 提交时能判断太轻、无绑点、无抓地。

### 16.2 第 2 周：下锚模拟

- 完成入水判定。
- 完成下沉阶段。
- 完成边框贴合面积衰减。
- 完成部件断开。

验收标准：

- 同一结构在不同海况下有不同结果。
- 弱连接会先损伤再断开。
- 太轻结构会漂浮。

### 16.3 第 3 周：触底与结算

- 完成海底抓地。
- 完成船绳拉力。
- 完成最终稳定判定。
- 完成失败原因归因。

验收标准：

- 玩家能理解为什么成功或失败。
- 至少 10 个关卡可玩。
- 成功率可通过配置调到合理区间。

### 16.4 第 4 周：表现与调优

- 增加水花、断裂、抓地表现。
- 增加材料图标和风险 UI。
- 调整参数手感。
- 优化判定间隔。

验收标准：

- 单局 1-3 分钟。
- 低端设备连接数量 50 内稳定运行。
- 玩家在 3 局内理解核心规则。

## 17. 主要风险与解决方案

### 17.1 物理不可控

风险：真实物理太复杂，结果难调。

解决：

- 一期采用阶段式半模拟。
- 关键结果由公式控制，物理表现做辅助。
- 连接断裂用面积衰减表现，而不是完全依赖引擎 joint。

### 17.2 玩家看不懂失败原因

风险：玩家觉得系统在乱判。

解决：

- 拼装阶段给风险等级。
- 下锚阶段高亮危险连接。
- 结算展示前三个失败原因。

### 17.3 拼装自由度过高导致性能差

风险：物品太多、连接太多，计算卡顿。

解决：

- 一期限制材料数量。
- 限制有效连接上限。
- 连接判定间隔化。
- 小碎片做视觉装饰，不参与受力。

### 17.4 最优解单一

风险：玩家只堆重物。

解决：

- 过重增加绳索拉力和入水冲击。
- 水阻、抓地、连接强度共同决定结果。
- 不同海底类型鼓励不同结构。

## 19. 可玩原型验收清单

- 玩家能看到本局船重、海况、水深、海底类型。
- 玩家能拖拽、旋转、叠放材料。
- 边框贴合后能生成连接。
- 材料重量会影响是否下沉。
- 材料粘合系数会影响连接强度。
- 材料会独立受力。
- 连接受力过大时，边框紧贴面积逐渐减小。
- 边框贴合面积过低后部件断开。
- 入水阶段会判断水面张力和浮力。
- 下沉阶段会计算水流横向力、船锚绳力、水阻和翻滚风险。
- 船绳会向绑点施加拉力。
- 水流会造成横向受力。
- 触底后会计算抓地力、海底摩擦和海底支撑。
- 最终根据锚固力和结构完整度结算。
- 结算能给出主要失败原因。

## 21. 一期设计结论

这个玩法的落地点不是追求完全真实的海洋物理，而是让玩家“拼装选择”能被系统清楚地检验。

一期应优先保证三件事：

1. 拼装阶段的连接规则清晰：2D 图形边框紧贴面积和材料决定粘合。
2. 下锚阶段的失败过程可见：受力、面积衰减、断裂都有表现。
3. 结算结果可解释：玩家能知道下次应该加重、加大边框贴合面积、换抓地件或强化绑点。

只要这三点成立，后续再扩展更多材料、海况和关卡，玩法就能自然长出来。
