# 《断锚求生》2D 拼装船锚玩法策划案

版本：v0.2  
日期：2026-07-03  
定位：2D 物理拼装 + 阶段受力验证游戏

---

## 1. 核心概念

### 1.1 一句话玩法

海上船锚突然断裂，船持续受到横向外力并向远处危险区漂移。玩家在限定时间内用一网捞上来的杂物拼出临时船锚，下锚后通过绳子拖住船；如果船在目标时间内没有漂到危险区，则本局成功。

### 1.2 玩法关键词

- 2D 侧视角。
- 随机材料。
- 限时拼装。
- 图形边框贴合连接。
- 材料重量与粘合系数。
- 船锚绳拉力。
- 水流横向力。
- 入水冲击。
- 海底摩擦与支撑。
- 可解释失败。

### 1.3 一期目标

一期目标是做出一个可玩的 2D 原型，而不是做真实海洋物理模拟。

玩家的主要决策是：

- 选哪些材料做主体。
- 哪些材料放在底部承受入水冲击。
- 哪些材料做抓地和摩擦。
- 船锚绳绑点放在哪里。
- 如何通过边框贴合获得更稳定的连接。
- 如何让船锚在足够长时间内减缓船的横向漂移。

---

## 2. 每局流程

1. 开局随机生成船只重量、海况、水深、海底类型。
2. 系统随机给出一批材料。
3. 玩家在限定时间内拼装船锚。
4. 玩家选择船锚绳绑点。
5. 系统检查结构是否满足基础下限。
6. 船锚进入入水阶段。
7. 船锚进入下沉阶段。
8. 船锚进入触底阶段。
9. 系统结算成功、勉强成功或失败。
10. 结算界面给出主要失败原因。

## 3. 开局随机参数

| 参数 | 说明 | 一期建议 |
|---|---|---|
| 船只重量 | 决定船锚绳拉力与最终稳船需求 | 800kg - 5000kg |
| 海况等级 | 影响入水冲击、水流、船锚绳波动 | 1 - 5 |
| 水深 | 决定下沉持续时间 | 8m - 40m |
| 海底类型 | 决定摩擦、支撑和抓地表现 | 沙地、泥地、碎石、岩底 |
| 拼装时间 | 玩家操作时间 | 45s - 120s |
| 材料数量 | 本局捞上来的物品数量 | 8 - 12 |
| 船体迎流系数 | 水流对船体拖拽强度 | 低、中、高 |
| 危险区距离 | 船从初始位置漂到危险区边界的距离 | 20m - 80m |
| 稳船目标时间 | 船需要坚持不进危险区的时间 | 20s - 60s |

## 4. 材料系统

### 4.1 材料核心属性

| 属性 | 作用 |
|---|---|
| 重量 | 影响下沉、入水冲击惯性、触底压力 |
| 密度 | 影响浮力 |
| 粘合系数 | 影响边框贴合后的连接强度 |
| 抗拉强度 | 抵抗船锚绳拉扯 |
| 抗剪强度 | 抵抗水流横向剪切 |
| 水阻系数 | 影响下沉阶段受水流影响 |
| 摩擦系数 | 影响触底后的抗滑能力 |
| 支撑稳定系数 | 影响触底后是否容易翻倒 |
| 抓地系数 | 钩形、尖锐、不规则物体的抓地能力 |

### 4.2 图形与碰撞箱分离

每个物品至少配置三层数据。

| 数据层 | 用途 |
|---|---|
| 显示图形 | 玩家看到的 2D 外观 |
| 拼装边框 | 用于判断能否贴合连接 |
| 实际碰撞箱 | 用于物理碰撞、受力、触底支撑 |

规则：

- 显示图形可以有破损、锈蚀、弯曲细节。
- 拼装边框用于连接判定，通常比显示图形更规整。
- 实际碰撞箱用于模拟阶段，通常比显示图形更简化。
- 玩家操作反馈必须以拼装边框为准，避免玩家觉得“看着贴上了但没连上”。

### 4.3 材料示例

| 材料 | 定位 |
|---|---|
| 生锈铁块 | 重，适合做主体，下沉强 |
| 破木板 | 边框长，易贴合，但轻 |
| 渔网团 | 可做缓冲或辅助连接，抗拉弱 |
| 铁钩 | 抓地强，适合触底端 |
| 石块 | 重，不规则，贴合不稳定 |
| 船链碎段 | 可传递拉力，但局部受力高 |
| 橡胶轮胎 | 缓冲强，水阻大 |
| 破铁锚爪 | 稀有材料，抓地强 |
| 塑料桶 | 漂浮，容易干扰下沉 |
| 水泥块 | 重，触底支撑稳定 |

## 5. 拼装阶段

### 5.1 拼装目标

玩家需要拼出一个满足以下条件的临时船锚：

- 总重量足够下沉。
- 绑点能连到主体。
- 主体连接足够牢。
- 底部材料能承受入水冲击。
- 触底后有足够摩擦、支撑或抓地能力。

### 5.2 操作方式

- 拖拽物品进入拼装区。
- 支持移动、旋转、翻转。
- 物品通过边框靠近和贴合生成连接。
- 不鼓励大面积重叠。
- 玩家选择一个物品作为船锚绳绑点。
- 可撤销、删除、重新摆放。

### 5.3 边框贴合连接

本方案不使用重叠面积作为连接依据。  
连接依据改为 2D 图形边框的贴合长度。

内部计算建议用“有效贴合长度”；对玩家展示时可以叫“贴合面积”或“接触面”。

连接生成条件：

| 条件 | 说明 |
|---|---|
| 边框距离 | 两个物品的拼装边框距离小于冗余距离 |
| 边框角度 | 两段边框角度接近，或能形成稳定卡靠 |
| 贴合长度 | 有效贴合长度达到最小值 |
| 材料兼容 | 两个材料允许连接 |
| 穿模限制 | 实际碰撞箱不能穿透过深 |

### 5.4 冗余空间

贴合不要求碰撞箱真正接触。  
系统允许一定冗余空间，避免像素级操作。

推荐判定：

```text
如果 边框最短距离 <= 贴合冗余距离
且 边框夹角 <= 最大贴合角度
且 有效贴合长度 >= 最小贴合长度
且 碰撞箱穿透 <= 最大允许穿透
则 生成连接
```

推荐参数：

| 参数 | 默认值 |
|---|---:|
| 贴合冗余距离 | 8 px |
| 最大贴合角度 | 18 度 |
| 最小贴合长度 | 16 px |
| 最大允许轻微穿透 | 6 px |
| 穿模禁止阈值 | 18 px |

### 5.5 连接强度

```text
连接强度 = 有效贴合长度
        * 材料A粘合系数
        * 材料B粘合系数
        * 表面粗糙度修正
        * 边框角度修正
        * 压靠修正
```

边框关系修正：

| 关系 | 修正 |
|---|---:|
| 平行贴合 | 1.00 |
| 小角度贴合 | 0.85 |
| 十字压靠 | 0.65 |
| 点接触 | 0.30 |
| 穿模过深 | 0，不生成连接 |

### 5.6 连接受力、位移与损伤

连接受力后不要立即散架。每条连接先允许两个材料产生一定移动量，包括拉开、滑移、旋转错位。

连接状态分为 4 层：

| 状态 | 表现 | 逻辑 |
|---|---|---|
| 稳定 | 连接边正常显示 | 受力低于连接强度 |
| 拉伸 | 两个材料出现轻微相对位移 | 消耗弹性位移，不降低贴合长度 |
| 滑移 | 连接边发黄或抖动 | 有效贴合长度开始下降 |
| 断开 | 材料脱离主体 | 有效贴合长度过低或位移过大 |

```text
连接相对位移 += max(0, 当前受力 - 当前连接强度) * 位移系数 * dt
```

当连接相对位移超过弹性位移上限后，才开始损失有效贴合长度。

```text
贴合长度损失 = max(0, 连接相对位移 - 弹性位移上限)
            * 贴合损伤系数
            * dt
```

断开条件：

```text
当前有效贴合长度 < 最小贴合长度
或 当前有效贴合长度 <= 初始贴合长度 * 最小保留比例
或 连接相对位移 >= 最大允许位移
```

阶段差异：

- 入水阶段允许强冲击直接把弱连接冲开。
- 下沉阶段以持续位移、滑移、贴合长度下降为主，不应一受力就直接散架。
- 触底阶段根据摩擦、支撑和船锚绳持续拉力继续累积位移与损伤。

推荐参数：

| 参数 | 默认值 | 说明 |
|---|---:|---|
| elastic_displacement_limit | 4 px | 弹性位移上限 |
| max_joint_displacement | 18 px | 最大允许位移 |
| displacement_gain | 0.12 | 受力转位移系数 |
| attach_length_decay | 0.08 | 贴合长度损伤系数 |
| sink_break_delay | 0.6s | 下沉阶段最短断开延迟 |

### 5.7 提交风险提示

提交下锚不做强硬阻挡。玩家只要愿意就可以提交，让结构进入入水、下沉、触底验证。

提交时系统只做风险提示，不禁止下锚：

| 提示项 | 触发条件 | 提示文案方向 |
|---|---|---|
| 主体未连通 | 有材料没有连到绑点主体 | 有些材料没有接进主体，可能下水就脱落 |
| 有拼接但断链 | 存在边框连接，但连接图不是一个整体 | 拼起来的部分没有全部连在一起 |
| 绑点风险 | 绑点到主体的承力路径太少 | 绳子可能先把绑点拉掉 |
| 重量风险 | 总重量明显偏低 | 可能压不进水或下沉很慢 |
| 底部风险 | 底部材料连接弱或质量轻 | 入水冲击可能把底部冲开 |
| 触底风险 | 摩擦、支撑、抓地能力低 | 到海底后可能拖不住船 |

提示规则：

- 如果所有已拼接材料都在同一连接图中，显示“主体连通”。
- 如果存在未连通材料，高亮未连通材料和断链位置。
- 如果结构明显危险，只显示风险等级和原因，不禁用“下锚”按钮。
- 允许玩家故意提交很烂的结构，以获得物理反馈和失败演出。

## 6. 受力阶段总览

下锚后只有三个阶段：

1. 入水
2. 下沉
3. 触底

不再拆成水面张力、船锚绳、独立水流、最终沉底等多个阶段。  
船锚绳力、水流横向力、重力等作为下沉或触底阶段内的受力项处理。

---

## 7. 入水阶段

### 7.1 阶段目标

检验船锚刚入水时，底部材料和底部连接能否承受水面对船锚下方的短时大冲力。

### 7.2 入水冲力

当船锚底部接触水面时，从船锚下方对底部材料施加向上的短时冲力。

```text
入水冲力 = 入水基础冲力
        * 海况修正
        * 下落速度修正
        * 底部投影宽度修正
```

推荐持续时间：0.15s - 0.4s。

### 7.3 受力对象

入水冲力优先作用于：

- 最下方材料。
- 最先接触水面的材料。
- 底部投影宽度最大的材料。

冲力传导方式：

- 底部材料受到向上冲击。
- 上方材料因惯性继续向下。
- 底部连接承受拉力与剪切力。
- 入水阶段允许出现强烈结果：弱连接可以被直接冲开，底部材料可以被水面冲散。

### 7.4 入水失败

| 失败 | 条件 |
|---|---|
| 底部被冲散 | 底部关键连接断开 |
| 结构弹起 | 总重量不足且浮力过高 |
| 绑点初裂 | 初始绳力叠加入水冲击导致绑点路径断裂 |

---

## 8. 下沉阶段

### 8.1 阶段目标

船锚进入水下后持续下沉，模拟重力、浮力、水流横向力、船锚绳拉力和水阻对结构的影响。

下沉阶段强调持续拉扯和结构变形：材料需要先产生一定移动量，再逐步损伤连接，不应一受力就直接散架。
### 8.2 主要受力

| 受力 | 影响 |
|---|---|
| 重力 | 让船锚下沉，重物拉扯连接 |
| 浮力 | 抵消下沉，轻质物上浮 |
| 水阻 | 减慢下沉，造成姿态变化 |
| 水流横向力 | 造成横向偏移和剪切损伤 |
| 船锚绳拉力 | 从绑点拉扯整个结构 |

### 8.3 下沉速度

```text
净下沉力 = 总重力 - 总浮力 - 纵向水阻
下沉速度 += 净下沉力 / 有效质量 * dt
```

### 8.4 水流横向力

```text
水流横向力 = 水流速度²
          * 部件横向投影
          * 部件水阻系数
          * 海况修正
```

水流横向力主要造成：

- 连接剪切损伤。
- 结构横向偏移。
- 大面积轻质材料拖拽主体。
- 姿态翻转风险。

### 8.5 船锚绳拉力

```text
船锚绳拉力 = 船重修正
          + 船体迎流修正
          + 海况波动修正
```

传导规则：

- 先作用于绑点材料。
- 沿连接图向主体传导。
- 贴合长度短的连接更容易损伤。
- 承力路径少时，单条连接受力更高。

### 8.6 部件独立受力

每个材料独立计算受力。  
如果某个材料与绑点主体断开：

- 不再参与最终稳船计算。
- 重物继续下沉。
- 轻物上浮或悬浮。
- 脱落过程作为失败表现。

### 8.7 下沉失败

| 失败 | 条件 |
|---|---|
| 主体逐步散架 | 有效主体在持续位移和连接损伤后低于总重量 50% |
| 绑点断开 | 船锚绳绑点脱离主体 |
| 水流撕裂 | 横向剪切先造成滑移和贴合长度下降，随后关键连接断开 |
| 下沉不足 | 浮力和水阻导致无法到达海底 |

---

## 9. 触底阶段

### 9.1 阶段目标

船锚接触海底后，水流横向力降低，同时增加海底摩擦与海底支撑。该阶段不只判断船锚本身是否稳定，还要持续模拟船受到横向力漂移、船锚通过绳子反向拉船的过程。

### 9.2 阶段变化

进入触底后：

- 水流横向力减少。
- 材料获得海底支撑。
- 材料与海底产生摩擦。
- 钩形、尖锐、不规则部件产生抓地力。
- 船锚绳拉力继续拖拽结构。

```text
触底水流横向力 = 下沉阶段水流横向力 * 触底水流衰减系数
```

推荐衰减：

| 海底 | 衰减系数 |
|---|---:|
| 泥地 | 0.35 |
| 沙地 | 0.45 |
| 碎石 | 0.55 |
| 岩底 | 0.65 |

### 9.3 海底支撑

```text
支撑力 = 接触材料重量
      * 海底支撑系数
      * 接触稳定系数
```

支撑影响：

- 防止继续下落。
- 决定是否翻倒。
- 决定抓地部件是否保持有效角度。

### 9.4 海底摩擦

```text
摩擦力 = 法向压力
      * 材料摩擦系数
      * 海底摩擦系数
```

法向压力来自：

- 材料重量。
- 船锚绳向下分量。
- 触底后的结构压靠。

### 9.5 抓地力

```text
抓地力 = 材料抓地系数
      * 海底匹配系数
      * 接触角度系数
      * 压入系数
```

### 9.6 稳定判定

触底阶段持续模拟到稳船目标时间结束，或船进入危险区，或船锚结构失效。

```text
船横向漂移力 = 船体横向外力 + 船体受流力
船锚反拉力 = min(船锚绳可承受拉力, 船锚触底抗拖拽能力)
船净漂移力 = 船横向漂移力 - 船锚反拉力
船漂移速度 += 船净漂移力 / 船体有效质量 * dt
船位置 += 船漂移速度 * dt
```

成功条件：

```text
船位置 < 危险区边界
且 已坚持时间 >= 稳船目标时间
且 船锚绳未断开
且 绑点主体未断开
```

### 9.7 触底失败

| 失败 | 条件 |
|---|---|
| 船进危险区 | 船锚反拉力不足，船在目标时间内漂入危险区 |
| 锚被拖走 | 船锚触底抗拖拽能力不足，无法继续提供反拉力 |
| 翻倒失效 | 支撑不稳，抓地角度失效 |
| 绑点拉断 | 持续绳力超过连接路径强度 |
| 底部滑移 | 摩擦不足，整体横向滑动 |

## 10. 结算

### 10.1 结算档位

| 档位 | 条件 |
|---|---|
| 完美稳住 | 船坚持到目标时间，且距离危险区仍有较大余量，主体损伤低于 20% |
| 勉强稳住 | 船坚持到目标时间，但接近危险区或主体损伤较高 |
| 入水失败 | 入水冲击导致结构崩坏或弹起 |
| 下沉失败 | 下沉中主体逐步散架、绑点断开或无法到达海底 |
| 触底失败 | 触底后船锚被拖走、翻倒、滑移，无法继续拉住船 |
| 船只漂失 | 船在目标时间结束前进入危险区 |

### 10.2 失败原因

最多显示 3 条：

- 总重量不足。
- 底部材料抗冲击不足。
- 边框贴合长度太短。
- 贴合角度不稳定。
- 绑点承力路径太弱。
- 材料粘合系数太低。
- 水流横向力过大。
- 触底摩擦不足。
- 海底支撑不足。
- 抓地角度不对。

## 13. 实现建议

### 13.1 边框检测

- 拼装边框使用多段线。
- 每个物品配置若干可连接边段。
- 拖拽时只检测当前物品与附近物品。
- 松手时计算最终有效贴合长度。
- 如果碰撞箱穿透超过阈值，禁止生成连接。

### 13.2 性能优化

- 普通连接每 0.1s 更新一次受力损伤。
- 高风险连接每帧更新。
- 一期有效连接上限控制在 50 条以内。
- 小碎片可只做视觉，不参与完整受力。

## 15. 验收清单

- 游戏明确为 2D。
- 物品区分显示图形、拼装边框、实际碰撞箱。
- 连接判定使用边框贴合长度，不使用重叠面积。
- 贴合判定有冗余空间，不要求碰撞箱严格接触。
- 穿模过深不能生成连接。
- 入水阶段从船锚下方给底部材料短时大冲力。
- 下沉阶段包含重力、浮力、水流横向力、船锚绳拉力。
- 触底阶段水流横向力减少，并增加海底摩擦和海底支撑。
- 材料独立受力。
- 连接受力过大时有效贴合长度下降。
- 最终结算能解释失败原因。
- 船会受到横向力向危险区漂移。
- 船锚会通过绳子对船施加反拉力。
- 船在目标时间内没有进入危险区才算成功。

## 16. 设计结论

这个玩法的一期核心不是复杂真实物理，而是让玩家拼装出来的 2D 船锚结构参与一场明确的稳船目标：船受到横向力向危险区漂移，船锚通过绳子把船拉住，只要船在目标时间内没有进入危险区就算成功。

关键落地点：

1. 拼装连接从“重叠面积”改为“边框贴合长度”。
2. 判定允许冗余空间，操作不需要像素级贴合。
3. 显示图形、拼装边框、实际碰撞箱必须分离。
4. 下锚模拟只保留入水、下沉、触底三个阶段。
5. 每个阶段都要让玩家看懂失败发生在哪里。
